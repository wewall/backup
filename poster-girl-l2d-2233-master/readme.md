# poster-girl-l2d-2233
2233娘的 Live2D 看板娘插件(WordPress)！

[Demo: https://www.fczbl.vip/946.html](https://www.fczbl.vip/946.html)

## 使用方法
1. Star 本项目

2. Download ZIP, 解压

3. 上传至站点 `/wp-content/plugins` 目录下

4. 登录后台启用即可

## 使用须知
插件需要jQuery和Font Awesome的支持，如果你的主题没有这两个东西，请去插件设置页面启用一下
